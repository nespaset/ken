# Ken (kn)

Минималистичный хелпер для Arch Linux и AUR, написанный на Go.

## Возможности

- Установка пакетов из официальных репозиториев и AUR одной командой.
- Автоматическое разрешение и установка зависимостей.
- Режим работы без пароля (настройка через visudo).
- Быстрый поиск по AUR.
- Автоматическая проверка обновлений через GitHub API.

## Использование

ken/kn [пакет]      - Установить пакет (сначала поиск в репо, затем в AUR).

ken/kn u            - Полное обновление системы (pacman -Syyu).

ken/kn r [пакет]    - Удаление пакета и его зависимостей (Rns).

ken/kn s [запрос]   - Поиск пакетов в AUR.

ken/kn visudo       - Настройка разрешений для работы без ввода пароля sudo.

ken/kn y [пакет]    - Установка с автоматическим подтверждением всех действий.

ken/kn n [пакет]    - Установка без проверки зависимостей (nodeps).

## Установка

1. Установите зависимости:
   sudo pacman -S go git

2. Склонируйте репозиторий:
   git clone https://github.com/nespaset/ken/
   
   cd sake

4. Скомпилируйте и установите:
   go build -o ken main.go
   
   sudo mv ken /usr/local/bin/

6. (Опционально) Настройте беспарольный режим:
   ken visudo

## Обновление

Программа автоматически проверяет наличие новых коммитов в ветке main. 
Для срабатывания триггера обновления текст коммита должен содержать слово "update".
